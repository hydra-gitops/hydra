# hydra ci config

CI configuration management.

## Synopsis

```bash
hydra ci config show
hydra ci config validate
```

## Description

Manages `.hydra-ci.yaml` configuration:
- **`config show`** — Display effective CI configuration
- **`config validate`** — Validate CI config syntax and references

## Examples

```bash
hydra ci config show
hydra ci config validate
```
